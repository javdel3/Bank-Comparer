## Program Description:
Choose from more than 100 Banks to compare which bank will the return on your money.
## Instructions:
1) Download the files for the program
2) Open the terminal and type ``g++ *,cpp -o BankComparerApp`` and press ENTER
3) Then type ``./BankComparerApp`` and press ENTER to start the program
